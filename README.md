## 📌 Project Overview

This project presents an **interactive Sales Dashboard** built using **Power BI** to analyze business performance for the year 2026.
It provides insights into **sales trends, revenue, and key performance indicators (KPIs)** to support data-driven decision-making.

---

## 🎯 Objectives

✔ Monitor overall sales performance
✔ Identify top-performing products and regions
✔ Track monthly and yearly sales trends
✔ Analyze revenue distribution
✔ Generate actionable business insights

---

## 🛠️ Tools & Technologies

* 📊 Power BI
* 🔄 Power Query (Data Transformation)
* 🧮 DAX (Data Modeling & Measures)
* 📁 Excel / CSV Dataset
